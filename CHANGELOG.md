## v05 - List Editing and Color Selection Improvements

- Added horizontal scrolling support for tables.
- Added the ability to add records to the custom `Rank / Time / By` lists by entering a source rank.
- Added support for deleting the selected row from the custom lists.
- Automatically updated displayed rank values after adding or deleting records.
- Added Windows color picker support when clicking the color preview.
- Improved the RGB editing workflow by automatically filling RGB fields and applying the selected RGB color to the active segment.
- Improved ComboBox dropdown readability by using a readable black text color.
