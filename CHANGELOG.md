# Changelog

## v01 - Initial Version

- Created a desktop application for extracting Dedimania online records from TMNF Exchange.
- Added an embedded WebView2 browser.
- Added support for loading a track page inside the application.
- Extracted the online records table from the rendered DOM.
- Displayed rank, time, mode, player name, and server data in a grid.
- Added a segment details table for extracted text and style information.
- Added JSON export for extracted records.
